YouTube API/search client


В рамках проекта реализован клиент для работы с YouTube Data API, позволяющий выполнять поиск видео, получать дополнительную информацию о роликах и каналах, а также отображать её в удобном формате. Основной упор сделан на работу с REST API, обработку и преобразование данных (длительность видео, количество просмотров, дата публикации), а также на разделение бизнес-логики и UI (вынос форматирования данных в отдельные модули).


Стек: React, TypeScript, Vite, Fetch API, CSS Modules, YouTube Data API v3
